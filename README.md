# GenNERate: Leveraging Text Generation Models for Named Entity Recognition (NER)

Welcome to **GenNERate**, a repository dedicated to exploring and implementing text generation models for Named Entity Recognition (NER) tasks. This project aims to leverage the capabilities of large language models (LLMs) to improve NER performance and provide innovative approaches to entity extraction.
